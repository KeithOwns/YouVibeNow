---
name: chunking-architect
description: Automatically breaks down massive project goals into sequential, bite-sized tasks and populates the TASKS.md file.
---

## Instructions
1. Analyze the user's high-level goal or requirement document.
2. Break the massive goal down into a linear sequence of small, executable steps ("chunks").
3. Ensure the chunks are ordered logically (e.g., dependencies first, backend before frontend).
4. Do NOT execute any of the steps.
5. Open the local `TASKS.md` file and overwrite it with the newly generated step-by-step checklist using Markdown task list format (`- [ ]`).
6. Present the final checklist to the user for approval.
