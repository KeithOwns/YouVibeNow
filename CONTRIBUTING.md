# Contributing to YouVibeNow

First off, thank you for considering contributing to YouVibeNow! This repository is designed to be the open-source standard for "Vibe Coding" and AI prompting architectures, and community contributions are what will make it world-class.

## How Can I Contribute?

### 1. Submit a New Skill
The most valuable contribution you can make is submitting a custom automated workflow to the `skills/` library.
If you have written a `SKILL.md` file that automates a tedious process (e.g., automated PR generation, security auditing, database scaffolding), please share it!
1. Add your skill folder to `New Project folder/skills/`.
2. Ensure it has the correct YAML frontmatter.
3. Submit a Pull Request.

### 2. Improve the Curriculum
If you spot typos, confusing explanations, or have a suggestion for a new Lesson, open an Issue or submit a PR modifying the markdown files.

### 3. Translate
Help us bring the 3-Tier Hierarchy to the world by translating the curriculum into other languages.

## Pull Request Process
1. Fork the repo and create your branch from `main`.
2. If you've added code/skills, ensure they are tested and functional.
3. Update the `README.md` with details of changes to the interface, if applicable.
4. Issue that pull request!

## Code of Conduct
Please be respectful and collaborative. We are all figuring out this new era of AI software development together!
