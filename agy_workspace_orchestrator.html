A permanent blueprint of the PowerShell configurations used to stand up this AGY project

1
Create Local AGY Directory

Installs subfolders to lock the active agent workspace configuration inside this target folder.

mkdir .agents/skills; mkdir .agents/rules
2
Clone and Move Garry Tan gstack Skills

Pulls down the latest build-in agent configurations and relocates files straight into `.agents/`.

git clone https://github.com/garrytan/gstack.git C:\Users\admin\gstack_temp
3
Inject TDD Guardrails File

Generates custom workspace rules causing any activated agent to construct automated test scripts before building code.

Set-Content -Path .agents\rules\superpowers-tdd.md -Value "..."
4
Bootstrap Claude Memory Server

Applies background mapping so AGY boots memory tracking snapshots locally inside the workspace folder.

Set-Content -Path .agents\mcp_config.json -Value "{...}"
