Configuring ChatGPT (Free Version) on Desktop

Moving laterally from Android to your desktop browser: How to establish your Tier 1 Baseline Identity and Memory constraints using the free version of ChatGPT.

Conceptual Framing

Whether you are on a mobile device or a desktop computer running Windows, Linux, or macOS, the principle of Application-Level Global Instructions remains the same. You must define a permanent safety net and baseline identity that ChatGPT uses before you send any actual chat messages.

This tutorial covers how to access and set up Custom Instructions and Memory on the desktop browser version of ChatGPT (Free Tier).

1Access Settings in Your Browser

Open your preferred web browser and log in to ChatGPT:

  1. Navigate to chatgpt.com.
  2. In the bottom-left corner of the sidebar, click on your profile icon (your name or avatar).
  3. From the pop-up menu, select Customize ChatGPT (or Settings depending on the latest UI update).

2Configure Custom Instructions

Custom Instructions act as your absolute Tier 1 stateful prompts. The interface provides two boxes:

  1. "What would you like ChatGPT to know about you to provide better responses?"
    Use this to set your context. Example: "I am a web developer working mostly in JavaScript and Python. I prioritize security and clean architecture."
  2. "How would you like ChatGPT to respond?"
    Use this to set your Baseline Identity and Constraints. Example: "You are an expert, senior-level AI pair programmer. Always write in plain, direct English. Never use corporate buzzwords. Always provide complete code snippets without skipping lines for brevity."

Make sure the toggle for "Enable for new chats" is turned on.

3Manage Memory Settings

ChatGPT features a dynamic Memory system that remembers details across all your chats. This is a powerful feature for maintaining a cohesive agent persona.

  1. From your profile menu, go to Settings > Personalization.
  2. Ensure the toggle for Memory is turned on.
  3. To manage what the AI remembers, click Manage Memory. Here you can delete incorrect or outdated memories that might corrupt the AI's state over time.

Pro Tip: You can directly tell ChatGPT in the chat box to "Remember that I always use tailwindcss for styling," and it will add that to your global memory bank automatically.

4Cross-Platform Continuity

Because you are logging into the same OpenAI account, the Custom Instructions and Memory items you set here in the desktop browser will seamlessly sync with the Android app settings we configured in the previous step.

Your Tier 1 Application-Level Stateful Prompts are now unified across all devices!
